1. house_code:房子的唯一标识符

3. area:所在的区

5.  rent_type:租房类型

8. house_type:户型

14. longtitude:经度
15. latitude:维度
16. distance:距最近地铁距离
17. url:租房链接
18. avg_price:单位面积房价
